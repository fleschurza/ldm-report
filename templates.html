Platform differences — what to expect Claude gives the most natural, conversational output and follows the instructions most faithfully. Recommended for ongoing audit work and team use.

Gemini tends to be more formal and structured. It may inject platform-level disclaimers in your interface language regardless of instructions — this is a Gemini platform behaviour that cannot be overridden. Output quality is good but tone will be less conversational than Claude.

ChatGPT sits between the two. Note the ~8,000 character instruction limit — see the setup steps if your prompt gets truncated.

Setting up on Claude (Project)

  1. 1
    Create a new Project in Claude Go to claude.ai → Projects → New Project. Give it a name like "unmask.filter".
  2. 2
    Paste the system prompt into Project Instructions Open Project Settings → Project Instructions. Copy the system prompt below and paste it in. Claude has no character limit on project instructions.
  3. 3
    Upload the reference file to Project Knowledge In the same project, go to Project Knowledge → Add Content → Upload file. Upload unmask.filter_Core-Reference.txt. This is the single source of truth for all thresholds, legal citations, and case rules.
  4. 4
    Start auditing Open a new conversation inside the project and paste any text, document, or question. Conversation history is retained per project — useful for ongoing audit sessions across multiple conversations.
Claude Projects retain conversation history — ideal for ongoing audit work where continuity matters.

Setting up on Gemini (Gem)

  1. 1
    Create a new Gem Go to gemini.google.com → Gem Manager → New Gem. Name it "unmask.filter".
  2. 2
    Paste the system prompt into Instructions Copy the system prompt below and paste it into the "Instructions" field. This is the same shared prompt used across all platforms.
  3. 3
    Upload the reference file Click the attachment icon and upload unmask.filter_Core-Reference.txt. One file is all you need — it consolidates all source documents.
  4. 4
    Save and start auditing Click Save → Open the Gem. Paste any text or document directly into the chat window.
Gemini Gems are ideal for quick one-off audits — paste a paragraph, get an immediate plain-language verdict.

Setting up on ChatGPT (Custom GPT)

  1. 1
    Open GPT Builder Go to chat.openai.com → Explore GPTs → Create → Configure tab. Name it "unmask.filter".
  2. 2
    Paste the system prompt into Instructions Copy the prompt below and paste into the "Instructions" field. Note: ChatGPT has an ~8,000 character limit. If you hit it, remove the "HOW YOU OPERATE" section — it is fully covered by the reference file.
  3. 3
    Upload the reference file under Knowledge Scroll to the Knowledge section → Upload file → unmask.filter_Core-Reference.txt. Add this line at the end of your instructions: "When handling edge cases or needing specific thresholds and legal citations, search your knowledge file before responding."
  4. 4
    Disable web browsing (recommended) Under Capabilities, turn off "Web Browsing" unless you specifically want live data lookups. This keeps the auditor focused on the reference framework.
  5. 5
    Save and publish (to yourself or your team) Set visibility to "Only me" for private use or "Anyone with a link" to share with your team. Click Create.
ChatGPT has an ~8,000 character instruction limit. If your prompt is truncated, remove the "HOW YOU OPERATE" section — the reference file covers that logic in full.
