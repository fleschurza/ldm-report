Platform coverage

unmask.shield scans for harmful institutional language wherever you read or write it. This page tracks which platforms are supported, tested, and coming soon.

Last updated: 26 May 2026

✓ Tested Verified working by the team
~ Should work Code in place, awaiting test

Read detection — flags language in messages and documents you receive or open.
Compose detection — flags language as you type before you send or publish.
